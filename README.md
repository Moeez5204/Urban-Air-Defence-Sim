# Urban Air Defence Sim - Lahore 3D Defense Simulation

A 3D urban defense simulation system for Lahore city featuring topological analysis, threat detection, and autonomous drone defense coordination.

## Features

- 3D visualization of Lahore's urban environment
- Topological data analysis (Alpha Complex, Persistent Homology)
- Interactive Multiple Model (IMM) filtering for target tracking
- LSTM-based enemy movement prediction
- Adaptive Sector Defense Allocation (ASDA) for drone deployment
- Multi-drone radar network with communication sharing
- Post-mission movement analysis and replay


## Once installed run 
python main_lahore_3d.py
